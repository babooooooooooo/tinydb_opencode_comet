## 6. 已知限制

- 删除后空间不立即回收（无 compaction）
- B-tree 不实现重平衡合并
- GROUP BY 仅支持内存哈希
- 无 WAL（使用 Shadow Paging）
- 目录表使用 JSON 存储列定义
